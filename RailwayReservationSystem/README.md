# Railway Reservation System

A Java-based Railway Reservation System built using Core Java and File Handling. This project demonstrates Object-Oriented Programming (OOP), file handling, and menu-driven application development.

## Features

- Add Train
- View Trains
- Book Tickets
- Store Data in Text Files
- Console-Based User Interface
- Simple and Easy-to-Understand Project Structure

## Technologies Used

- Java 17
- File Handling
- Object-Oriented Programming (OOP)

## Project Structure

```
Main.java
RailwayService.java
Train.java
Passenger.java
Booking.java
trains.txt
bookings.txt
README.md
```
